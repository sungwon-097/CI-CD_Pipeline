### Ansible

-   IaC(Infrastructure as Code)
    -   시스템, 하드웨어 또는 인터페이스의 구성정보를 파일(스크립트)을 통해 관리 및 프로비저닝
    -   버전관리를 통한 리소스 관리(현재 상태를 History를 이용해 관리)
    -   할 수 있는 일
        1. 설치 : apt-get. yum, brew
        2. 파일 및 스크립트 배포 : copy
        3. 다운로드 : get_url, git
        4. 실행 : shell, task
